# discordjs-quickstart

A modular quickstart for a bot application written with discord.js

### Prerequisites

+ Nodejs

### Installing
```
git clone https://github.com/Madaoyasha/discordjs-quickstart
cd discordjs-quickstart
nano /src/config.json
npm i
node app.js
```
## Built With

* [Discord.js](https://discord.js.org/#/) - Discord framework
* [sqlite3](https://www.sqlite.org/index.html) - Database
